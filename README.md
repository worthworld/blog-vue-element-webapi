# blog-element-vue
   此项目基于vue-cli、vue-router、vuex、axios、element-ui、mavon-editor、Less的技术栈，且已开源；
   github地址：https://github.com/worthworld/blog-vue-element.git

#  特点
* 支持 MarkDown 编辑
* 支持移动端

[在线地址](https://www.stonemei.cn)

### 前端工具
* Vue2.0
* Vue-Router
* Vuex
* axios
* element ui
* Less
* mavon-editor
